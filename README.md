# Java Online Ordering System

A Java-based command-line ordering system built with object-oriented programming.

## Features

- Menu loading
- Order creation
- Inventory control
- Discount calculation
- Order lookup
- CSV-based data persistence

## Tech Stack

- Java
- Object-Oriented Programming
- File I/O
- CSV
